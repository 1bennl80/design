Project Gutenberg Website Redesign

The goal of this project was to research and design a new and improved website layout for the gutenberg project.
I started this project off by researching the current competitors and comparing them to the gutenberg's current website.
After this I began sketching ideas for the new layout and design. I tried to incoparate the original colours from the current design as best I could.
